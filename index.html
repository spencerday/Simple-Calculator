Simple Calculator

This calculator takes two numbers as operands, and lets you select which operation you want done on them.

Input

Operands can either be any number (i.e. 3, 12, 13.56, -23, etc), or you can spell out certain numbers. If you spell out the number it must be a number between 0 and 50 (inclusive), and has to be hypenated it there are two words (i.e. twenty-seven, thirty-two, etc).
